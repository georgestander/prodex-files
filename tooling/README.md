# tooling/

Local tooling configs and allowlists for this workspace.

- `tooling/new_session.sh` creates `sessions/<id>/` with `session.json` + `summary.md`.
- `tooling/new_entity.sh` creates `entities/<type>/<slug>/` with `entity.json` + `notes.md`.
- `tooling/codex_ps.sh` starts Codex in this repo and runs a read-only rehydration prompt.
